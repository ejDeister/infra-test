# Beer_Me
A fast, easy-to-use web app for tracking and organizing beers you’ve tried. You can quickly log beers, rate them, and keep everything in one place. It’s built to be simple and enjoyable, so you can build a personal beer library you'll love revisiting.

## Setup

- Clone to desired directory
- In new terminal enter 'npm install'
- Enter 'npm run start'
- in browser navigate to http://localhost:3000/
